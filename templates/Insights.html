Welcome to my insights page! Here I will share with you some of my thoughts and guides to educational content I have learnt. I hope you find something within this page that is insightful for you!



  • Harvard's CS50 Course


    • If you have minimal experience in computer science and coding, I highly recommend starting with Harvard's CS50: Introduction to Computer Science. In fact, this very website that you're on was made for Harvard's CS50 Final Project. After just 10 weeks of content, you too could be achieving this level proficiency or even more! ...

    It is an entry-level course taught by David J. Malan, it teaches students how to think algorithmically and solve problems efficiently. Topics include abstraction, algorithms, data structures, encapsulation, resource management, security, software engineering, and web development. Languages include C, Python, SQL, and JavaScript plus CSS and HTML. Problem sets inspired by real-world domains of biology, cryptography, finance, forensics, and gaming. You can sign up for the course for free and complete in at your own pace here is a link to sign up! Harvard CS50 sign up

    I will admit that the course was daunting at first as the jump from the lecture content to the problem set is fairly huge, but you will soon realise Google and Stackoverflow are your best friends to work through any bugs you face! Don't give up and I guarantee by week 5 of the course you will like programming and coding comes natural to you!

  • CS50 Final Project: Local Development, Git & GitHub


    • Throughout the course of CS50, students code in a virtual workplace called Codespaces. This is essentially accessing a virtual machine where all the local environments and packages has been set up for you by the CS50 team. This is great for new programmers as they do not have to deal with the hassle of setting up a workspace. However, I highly encourage you to attempt to develop your final project locally on your own local copy of VScode as it offers a truly rich learning experience. Feel free to refer to my GitHub repository where I show how to develop the week 9 problem set, Finance stock tracker, where I remove all CS50 training wheels and develop it locally: Finance Local Development ...

    I also firmly belive it is very important to learn about Git and Github and understand how to use it. In short it essentially tracks your changes locally (GIT) and allows you to post your changes to an online tracker (GITHUB) which u can then pull back into your local device. Please refer to my github repository where I discuss in detail further how to develop locally and use Git and Github. Some key topics included are: installing Windows Subsytem for Linux (WSL), install ubuntu to host WSL, learning about apt-get and pip3 packet managers, installing extensions, and all the resources and links needed to create a Github repository and how to utilise it with Git on your machine: Git & Github-More Local Developement

    I personally spent a lot of time learning how to develop locally and how to use Git & GitHub, I hope this guide flattens the learning curve for these topics which can be daunting for a new programmer, but don't worry it's really not as complicated as it appears to be!
