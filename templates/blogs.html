How I Started My Coding Journey

May 2023

My journey into computer science began with Harvard's CS50: Introduction to Computer Science in early 2023. In fact, this very website was built as my CS50 Final Project. That foundation launched me into building production applications like Aqua Vitae, working with microservices architectures in PeerPrep, and developing full-stack MERN applications. ...

If you're starting your coding journey, I highly recommend CS50. It's an entry-level course taught by David J. Malan that teaches you how to think algorithmically and solve problems efficiently. Topics include abstraction, algorithms, data structures, encapsulation, resource management, security, software engineering, and web development. Languages include C, Python, SQL, and JavaScript plus CSS and HTML. You can sign up for the course for free and complete it at your own pace: Harvard CS50 sign up

The course was challenging at first—the jump from lectures to problem sets is significant—but Google and Stack Overflow become your best friends. Don't give up! By week 5, programming will start to feel natural. From there, you can build real projects and continue growing your skills, just like I did.




Local Development, Git & GitHub Guide

May 2023

Throughout CS50, students code in Codespaces—a virtual machine with everything pre-configured. While great for beginners, I highly encourage developing your final project locally on your own machine. It offers a truly rich learning experience. Check out my GitHub repository where I show how to develop the CS50 Week 9 problem set (Finance stock tracker) locally without the training wheels: Finance Local Development ...

I also firmly believe it's important to learn Git and GitHub. In short, Git tracks your code changes locally (version control), while GitHub hosts your code online for backup, sharing, and collaboration. My GitHub repository covers everything you need: installing Windows Subsystem for Linux (WSL), setting up Ubuntu, using apt-get and pip3 package managers, installing VS Code extensions, and creating your first GitHub repository: Git & GitHub Guide

I personally spent a lot of time learning local development and Git/GitHub. I hope this guide flattens the learning curve for these topics, which can seem daunting for new programmers—but don't worry, it's really not as complicated as it appears!
